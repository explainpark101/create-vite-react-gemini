# create-vite-react-gemini

Vite + React + Tailwind CSS + Lucide React 프로젝트를 빠르게 생성하는 Bun create 스크립트입니다.

## 사용법

```bash
bun create vite-react-gemini <프로젝트명>
```

또는 현재 디렉터리에 생성:

```bash
bun create vite-react-gemini .
```

## 포함 사항

- **Vite** - React (JavaScript) 템플릿
- **Tailwind CSS** - `tailwindcss`, `@tailwindcss/vite` (최신 버전)
- **Lucide React** - 아이콘 라이브러리

## 생성 후 실행

```bash
cd <프로젝트명>
bun install
bun dev
```

## 로컬 개발/테스트

```bash
bun install
bun run index.ts my-app
```

## 기술 스택

- [Vite](https://vite.dev/) - 빌드 도구
- [React](https://react.dev/) - UI 라이브러리
- [Tailwind CSS](https://tailwindcss.com/) - 유틸리티 CSS
- [Lucide React](https://lucide.dev/) - 아이콘
